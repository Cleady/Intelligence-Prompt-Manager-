1. Cosine Similarity (Đại số tuyến tính)

Đo độ tương đồng giữa 2 vector từ khóa:

cos(θ) = (A·B) / (‖A‖ × ‖B‖)

→ Giá trị càng gần 1, prompt càng phù hợp với truy vấn

2. TF-IDF Vectorization

Biểu diễn văn bản thành vector số:

TF-IDF(t,d,D) = TF(t,d) × log(N/DF(t))

3. Weighted Scoring (Tối ưu hóa)

Kết hợp nhiều yếu tố với trọng số:

Score = w₁×sim_title + w₂×sim_content + w₃×recency

4. Gradient-inspired Ranking

Điều chỉnh thứ tự dựa trên "độ dốc" độ phù hợp:

Δrank ∝ ∂(relevance)/∂(features)
